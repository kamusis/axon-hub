# mmx search vs Exa — Comparison (2026-05-16)

Test query: "Hermes Agent by Nous Research latest features 2025"

## mmx search
- Returns 10 results
- **Chinese content dominant** (CSDN, cnblogs, QQ, 腾讯新闻)
- **Noise present:** "Hermes 2025春夏系列" (fashion), "爱马仕收入" (revenue) mixed in
- Snippets surface-level (introductory tutorials, rehashed articles)
- Good for: Chinese internet, domestic news, Chinese-language technical content
- Weakness: irrelevant results, shallow depth, no structured metadata

## Exa
- Returns N results (configurable)
- **All highly relevant** — GitHub releases, official site, docs, release notes
- **Rich content:** full release notes, feature lists, architecture details
- **Structured metadata:** stars, contributors, release dates, highlights
- Good for: English technical research, precise source discovery
- Weakness: limited Chinese internet coverage

## When to use which
- **mmx search:** Chinese language queries, domestic content, Chinese tech ecosystem
- **Exa:** English technical research, source-level accuracy, structured data needs
- **Both:** When comprehensive coverage matters across both ecosystems
